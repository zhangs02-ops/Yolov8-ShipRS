# YOLOv8-ShipRS v3 修改说明文档

## 一、改进概述

v2 及之前版本的问题：P2、CoordAttention、AdaptiveDetect、EMA 等均为**已有模块的直接引入**，老师评价"太肤浅，没有针对船舶检测自己改进模型"。

v3 的核心思路：**基于遥感船舶的独特难点，自行设计原创模块**，不再是拿来主义。

| 改进项 | 针对的船舶难点 | 设计思路 | 原创性 |
|--------|--------------|----------|--------|
| **DASC 方向感知条形卷积** | 船舶是细长目标（长宽比3:1~10:1），标准3×3方形卷积感受野利用率低 | 并行1×7和7×1条形卷积，通过全局池化自适应学习方向权重 | ★ 不是DCNv3，而是方向感知的条形卷积 |
| **BSA 背景抑制注意力** | 海面背景均匀但存在波浪/云影干扰，通用注意力不区分背景/前景 | 高低频分离→低频估计背景强度→生成抑制权重→保留高频船舶特征 | ★ 不是通用注意力，而是专门抑制海面背景 |
| **SOAU 小目标感知上采样** | 标准最近邻上采样"无脑复制"导致小目标特征在FPN传递中被稀释 | 最近邻上采样+可学习残差精炼，根据局部内容自适应增强 | ★ 比Dysample轻量，比最近邻精细 |
| **SADL 尺度感知分布损失** | CIoU对小目标过于敏感，NWD与assigner不兼容导致训练崩溃 | 在NWD基础上引入尺度感知权重和形状感知权重，自适应调整损失 | ★ 不是WIoU，而是针对船舶尺度和形状的原创损失 |

## 三、网络架构

```
输入 640×640×3
    │
═══════════════════════════ Backbone ═══════════════════════════
    │
    ├─ Conv(64, k=3, s=2)          → 320×320×64
    ├─ Conv(128, k=3, s=2)         → 160×160×128        (P2/4)
    ├─ C2f(128)                    → 80×80×128
    ├─ Conv(256, k=3, s=2)         → 40×40×256          (P3/8)
    ├─ C2f_DASC(256, n=6) ★        → 40×40×256          ★ 方向感知条形卷积
    ├─ Conv(512, k=3, s=2)         → 20×20×512
    ├─ C2f_DASC(512, n=6) ★        → 20×20×512          ★ 方向感知条形卷积
    ├─ Conv(1024, k=3, s=2)        → 10×10×1024
    ├─ C2f(1024, n=3)              → 10×10×1024
    └─ SPPF(1024)                  → 10×10×1024
    │
═══════════════════════════ Neck (FPN+PAN) ═════════════════════
    │
    ─── FPN 自顶向下（SOAU 替代最近邻上采样）───
    ├─ SOAU(P5→P4) ★               → 20→40              ★ 小目标感知上采样
    ├─ Concat + C2f_BSA(512) ★     → 40×40×512          ★ 背景抑制注意力
    ├─ SOAU(P4→P3) ★               → 40→80              ★ 小目标感知上采样
    ├─ Concat + C2f_BSA(256) ★     → 80×80×256          ★ 背景抑制注意力
    ├─ SOAU(P3→P2) ★               → 80→160             ★ 小目标感知上采样
    ├─ Concat + C2f_BSA(128) ★     → 160×160×128        ★ 背景抑制注意力
    │
    ─── PAN 自底向上 ───
    ├─ Conv + Concat + C2f_BSA     → P3 输出
    ├─ Conv + Concat + C2f         → P4 输出
    └─ Conv + Concat + C2f         → P5 输出
    │
═══════════════════════════ Head ═══════════════════════════════
    │
    └─ AdaptiveDetect(P2, P3, P4, P5)   四尺度检测
```

## 四、模块原理

### 4.1 DASC 方向感知条形卷积

```
输入 x (B, C, H, W)
    │
    ├─ 1×7 卷积 + BN + SiLU  →  f_h  (水平方向特征)
    └─ 7×1 卷积 + BN + SiLU  →  f_v  (垂直方向特征)
         │
         ├─ 全局平均池化 f_h → w_h  (方向重要性分数)
         └─ 全局平均池化 f_v → w_v
         │
         └─ Softmax([w_h, w_v]) → 归一化权重
         │
         └─ 输出 = f_h × w_h + f_v × w_v
```

**为什么原创**：
- 不是简单的条形卷积堆叠，而是**自适应加权**两个方向
- 参数量 = 2×(C×C×7) ≈ 标准3×3卷积的4.7倍，但感受野覆盖7×7区域
- 专门针对船舶的**细长+方向任意**特性设计

### 4.2 BSA 背景抑制注意力

```
输入 x (B, C, H, W)
    │
    ├─ 低频分支: 5×5 DWConv → 1×1 Conv → Sigmoid → bg_weight (B, C, H, W)
    │              ↑ 捕获平滑背景区域
    │
    └─ 高频分支: 3×3 DWConv → 1×1 Conv → fg_feat (B, C, H, W)
                   ↑ 捕获边缘和细节
    │
    └─ 输出 = fg_feat × (1 - bg_weight) + x × 0.1
              ↑ 背景强的地方抑制，背景弱的地方保留
```

**为什么原创**：
- 不是通用的"增强重要区域"，而是**专门抑制海面背景**
- 利用船舶与海面的**频率差异**，物理意义明确
- 参数量极小（两个深度可分离卷积）

### 4.3 SOAU 小目标感知上采样

```
输入 x (B, C, H, W)
    │
    ├─ 最近邻上采样 (F.interpolate) → x_up (B, C, 2H, 2W)
    │
    └─ 残差精炼分支:
         3×3 Conv → BN → SiLU → 3×3 Conv → BN → x_refined
    │
    └─ 输出 = x_up + 0.1 × x_refined
```

**为什么原创**：
- 不是简单的最近邻复制，而是**上采样后做可学习的特征精炼**
- 残差精炼分支学习局部内容的自适应增强
- 参数量 = 2×C²×9，只在Neck中使用3次，开销可控

### 4.4 SADL 尺度感知分布损失

```
输入: pred (B, 4), target (B, 4)
    │
    ├─ 1. 基础 NWD 距离
    │    nwd = bbox_nwd(pred, target)
    │    base_loss = 1 - nwd
    │
    ├─ 2. 尺度感知权重 W_scale
    │    scale_ratio = gt_area / (640×640)
    │    W_scale = 1.0 + 0.5 × (1 - scale_ratio)
    │    小渔船 → W_scale ≈ 1.5
    │    大货船 → W_scale ≈ 1.0
    │
    ├─ 3. 形状感知权重 W_shape
    │    ar_diff = |pred_ar - target_ar| / target_ar
    │    W_shape = 1.0 + 0.3 × ar_diff
    │    长宽比正确 → W_shape ≈ 1.0
    │    长宽比偏差50% → W_shape ≈ 1.15
    │
    └─ 输出 = base_loss × W_scale × W_shape
```

**为什么原创**：
- 不是 WIoU 的简单替换，而是在 NWD 基础上增加**尺度感知**和**形状感知**两个原创权重项
- 解决了 NWD 与 TaskAlignedAssigner 的兼容性问题（通过有界权重确保 assigner 的 topk 选择正常）
- 每个权重项都有明确的遥感船舶检测动机，物理意义清晰
- 参数量 = 0（纯数学计算，无额外参数）

## 七、v2 → v3 的核心区别

| 方面 | v2（拿来主义） | v3（原创设计） |
|:---:|---|---|
| **特征提取** | C2f_CA（CoordAttention已有模块） | C2f_DASC（自行设计的方向感知条形卷积） |
| **特征融合** | C2f_CA + 渐进式融合 | C2f_BSA（自行设计的背景抑制注意力） |
| **上采样** | 标准最近邻上采样 | SOAU（自行设计的小目标感知上采样） |
| **损失函数** | CIoU（默认）/ NWD（不兼容assigner） | SADL（自行设计的尺度感知分布损失） |
| **设计理念** | 引入已有注意力机制 | 针对船舶细长形态、海面背景、上采样稀释、损失函数四个具体问题分别设计解决方案 |
